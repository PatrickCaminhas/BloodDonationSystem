# **Blood Donation System**
A proposta deste projeto é uma aplicação para um sistema de doação de sangue

## Funcionalidades

É possivel nessa aplicação fazer um CRUD Completo para os seguintes segmentos:
- Tipos sanguineos
- Pessoas
- Locais de doações
- Doações


## Funcionalidades futuras

Será feito um CRUD Completo para cidades, estados, unidades, produtos e distribuições em adicional do que já foi feito para a disciplina.

## Tecnologias

- Node.js
- Typescript
- React


### Este projeto foi feito para disciplina de Sistemas WEB da Universidade Federal de Ouro Preto
